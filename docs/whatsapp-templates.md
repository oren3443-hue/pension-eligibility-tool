# תבניות WhatsApp לאישור (Glassix / Meta)

תיעוד של כל תבניות ה-WhatsApp שהאפליקציה שולחת. **כל אחת מהן חייבת אישור** ב-Glassix → WhatsApp Templates לפני שליחה לעובדים אמיתיים.

> שים לב: WhatsApp Business דורש placeholders ממוספרים (`{{1}}`, `{{2}}`...). באפליקציה אנחנו משתמשים בשמות (`{{first_name}}`, `{{eligibility_month}}` וכו'). הקוד מרנדר את הטקסט המלא לפני השליחה, ו-Glassix משווה לתבנית המאושרת לפי תוכן ה-body. **חשוב: התאמה חייבת להיות בייט-לבייט** — כל שינוי קוסמטי בקוד (רווח, אימוג'י, פסיק) חייב לעבור אישור מחדש.

## איך לאשר ב-Glassix

`Settings → WhatsApp Templates → New Template` → מזין את ה-fields למטה לכל אחת מהתבניות. Meta מאשרת בד"כ תוך 24-48 שעות. אם נדחה — Glassix יראה את הסיבה.

## 1. `pension_start_contribution` — תחילת הפרשה (הודעה א)

| שדה | ערך |
|---|---|
| Template Name | `pension_start_contribution` |
| Category | `UTILITY` |
| Language | Hebrew (`he`) |
| Header | (ריק) |
| Footer | (ריק) |
| Buttons | (ריקים) |

**Body:**

```
שלום {{1}} 👋

אנו שמחים לעדכן כי החל מחודש {{2}} נתחיל בביצוע הפרשות פנסיוניות עבורך.

מה עליך לעשות?

באפשרותך לבחור את הקופה אליה תרצה שנפקיד את הכספים. את פרטי הקופה יש לשלוח למייל: payroll@orenmeshi.com.

שימי לב: יש להעביר את הפרטים לא יאוחר מה-{{3}}.

במידה ולא יועברו פרטים עד למועד זה, ההפקדות יועברו באופן אוטומטי לקופת ברירת המחדל שתבחר עבורך, כדי לשמור על זכויותיך.

בברכה,
מחלקת שכר, אורן משי 🩵
```

**Sample values:**
- `{{1}}` = `אורנה`
- `{{2}}` = `יוני 2026`
- `{{3}}` = `15 ביוני 2026`

**מיפוי לקוד האפליקציה:**
- `{{1}}` ⇄ `{{first_name}}`
- `{{2}}` ⇄ `{{eligibility_month}}`
- `{{3}}` ⇄ `{{deadline_date}}`

## אזהרות

### Emoji
- 👋 (Template 1) ו-🩵 (כל השלושה) — Meta בד"כ מאשרת. אם נדחה: להחליף ל-טקסט פשוט (`שלום` בלי 👋, ו-`בברכה,` בלי 🩵).

### שינויים בעתיד
כל שינוי בנוסח (גם הוספת רווח, החלפת פסיק, הסרת אימוג'י) דורש **אישור מחדש**. תהליך מומלץ:
1. לערוך את ה-template ב-`src/App.tsx` (`TEMPLATE_PRESET_*`)
2. לעדכן את ה-body כאן בתיעוד
3. ליצור גרסה חדשה ב-Glassix (`pension_start_contribution_v2` וכו')
4. לחכות לאישור מ-Meta
5. רק אחרי שהגרסה המעודכנת מאושרת — לדחוף לאתר

### המייל
`payroll@orenmeshi.com` קשיח (hard-coded) בכל שלושת ה-templates. אם המייל משתנה — צריך לעדכן ב-3 מקומות + לאשר מחדש.
